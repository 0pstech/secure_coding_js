## 실습 환경
- node version 22.14.0
- visual code
- mysql
- express
- nestjs

## 설치 필요
- docker (docker-compose)
- nvm install 22.14.0
- nvm use 22.14.0
- npm install -g ts-node typescript @types/node

## 실행 방법
 - js: node src.js
 - ts: ts-node src.ts
